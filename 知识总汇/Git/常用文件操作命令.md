### 进入某个目录
```
 cd xx 
```

### 回退上级目录
```
cd..
```

### 创建一个目录
```
 mkdir xxx 
```

### 创建一个文件
```
 touch xx.xx 
```

### 删除一个文件
```
 rm -f xx.xx 
```

### 显示当前目录
```
 pwd 
```
### 查看目录及子目录的所有文件
```
tree
```

### 查看当前文件夹目录文件及文件夹
```
ll
```

### 清除屏幕信息
```
clear
```

### 查看历史操作命令
```
history
```

### 退出DOS窗口
```
exit
``
