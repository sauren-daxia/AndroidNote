LinkedList：	List的子类
-------------------------------
添加：
void		addFirst()	//添加元素到第一位
void		addLast()	//添加元素到最后一位，和add没区别
-------------------------------
获取：
Object		getFirst()	//获取第一位元素
Object		getLast()	//获取最后一位元素
-------------------------------
删除：
Object		removeFirst()	//删除第一位元素并返回它
Object		removeLast()	//删除最后一位元素并返回它
-------------------------------
一般方法参考List
