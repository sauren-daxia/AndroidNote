-----------------------------------------------------
<?>:		//在泛型类中使用，作用是任意类型。
使用方法：
Collection<?>	c = new Collection<Student>();		//可以先不确定指定类型，在创建的时候再指定，会灵活很多
-----------------------------------------------------
<? extends E>	//在泛型类中使用，作用是任意E的子类
使用方法：
Collection<? extends Animal> c = new Collection<Dog>();	//限定只能是自身或继承自animal的子类才能创建
-----------------------------------------------------
<? super E>	//在泛型类中使用，作用是任意E的父类
使用方法：
Collection<? super Dog> c = new Collection<Object>	//限定只能是自身或自身的父类
