ReadWriteLock	读写锁

创建读写锁:
--------------------------------------------------
ReadWriteLock lok = new ReentrantReadWriteLock();

获取读锁：
--------------------------------------------------
Lock		readLock().lock()	//获取锁同时调用lock()锁上

获取写锁：
--------------------------------------------------
Lock		writeLock().lock()	//获取锁同时调用lock()锁上

释放锁：
--------------------------------------------------
void		writeLock().unlock()	//写完之后再拿到锁，同时再调用unlock()解锁

void		readLock().unlock()	//写完之后再拿到锁，同时再调用unlock()解锁

--------------------------------------------------
PS:不能同时写入，不能边写边度，但能同时读取
